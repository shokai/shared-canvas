Shared Canvas
=============
shared canvas using html5 websocket, canvas.
[http://canvas.shokai.org](http://canvas.shokai.org)


Install Dependencies
--------------------

    % bundle install


Config
------

    % cp sample.config.yaml config.yaml

edit config.yaml.


Run
---

    % ruby development.ru
    % ruby websocket.rb


Deploy
------
use Passenger with "config.ru"
